Combine PDF

Aplikasi yang mudah digunakan ini memungkinkan Anda menggabungkan beberapa file PDF atau gambar menjadi satu dokumen PDF tanpa perlu menginstal software apa pun.

  1. Pilih file PDF dari komputer Anda, atau seret file tersebut ke area unggahan.
  2. Seret dan jatuhkan file untuk mengubah urutannya. Setelah siap, klik tombol GABUNGKAN.
